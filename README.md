Automação de Tarefas Front-end
Este projeto foi desenvolvido como atividade prática para criação e configuração de um Gulpfile capaz de automatizar tarefas essenciais no workflow de front-end.

As tarefas implementadas incluem:

Compilação de SASS → CSS
Compressão de imagens
Compressão e minificação de JavaScript
Geração de sourcemaps
Criação de estrutura organizada entre source/ e build/
